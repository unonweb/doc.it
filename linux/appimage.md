# FILES

You can put AppImages anywhere you want and run them from there -- even USB thumbdrives or network shares.

However, the official *recommendation* by the AppImage developers is to create an extra directory:

```sh
~/Applications/ # or
~/.local/bin/ # or
~/bin/
```
