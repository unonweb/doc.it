#### copy/change
`u`             # undo
`U`             # undo all changes on whole line
`ctrl + r`      # redo
`x`             # delete caracter („under“ the cursor and to the left)
`d + motion`    # delete 
`dd`            # delete line (2dd deletes two lines)  / dG # delete until end
