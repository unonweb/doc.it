# tar

```sh
tar -xzf

tar --extract # Extract files from an archive.  Arguments are optional.  When given, they specify names of the archive members to be extracted; tar -x
tar --gzip # Filter the archive through gzip; tar -z
tar --file=ARCHIVE # Use archive file or device ARCHIVE; tar -f
```

# compression options

```sh
tar --xz # Filter the archive through xz
```
